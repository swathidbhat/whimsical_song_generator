# VAN Mode Process Map

## IMMEDIATE RESPONSE
Respond: OK VAN

## VALIDATION & ASSESSMENT PROCESS

### Step 1: Gather Information
- Read user request completely
- Identify project type and scope
- Note constraints (time, team, tech)

### Step 2: Complexity Assessment
Determine VAN Level (1-4):
- **Level 1**: Simple, clear, direct implementation
- **Level 2**: Moderate complexity, needs planning
- **Level 3**: Complex, multiple systems
- **Level 4**: Highly complex, needs creative solutions

### Step 3: Create/Update Memory Bank
- projectbrief.md: Core concept, goals, constraints
- tasks.md: Break down into tasks
- activeContext.md: Current focus and status

### Step 4: Decision
- Level 1 → IMPLEMENT
- Level 2-4 → PLAN

### Step 5: Update Memory Bank
- Document VAN completion
- Set next mode
